Linux及Windows电源相关命令

Ubuntu和Windows的命令感觉大同小异

Ubuntu

现在关机
p=poweroff 

shutdown --r now

现在重启
r=reboot 

shutdown --r 3

3分钟后重启
在Ubuntu里数字的单位是分 

shutdown --p 20:00

在20：00关机

Windows

shutdown -s -t XX

XX秒后关机
-s=shutown;-t=time
XX是时间，单位是秒 

shutdown -r -t XX

XX秒后重启
-r=reboot 

shutdown -a

在进行命令后，还来得及的话用此命令取消之前的关机或重启命令
