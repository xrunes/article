配置使用You-get

You-get是一个在Github上开源的，基于Python的视频下载工具。官网在https://you-get.org/，食用方法。

You-get依赖Python和FFmpeg，需要解决这两个的依赖问题才能正常使用。

配置Python环境

在Python上下载Python，并且Add Python to Path。

安装FFmpeg

FFmpeg上下载并解压文件夹至根目录，然后在控制面板中的系统环境变量中添加/ffmpeg/bin目录的路径。

安装You-get

配置好Python和FFmpeg之后，在终端安装You-get
